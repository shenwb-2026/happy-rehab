# 快乐康复指导 — 真机测试部署说明

## 方式一：局域网测试（最简单，推荐）

### 1. 在本机启动开发服务器
```bash
cd happy-rehab
npm run dev
```
服务启动在 `http://10.250.39.42:3000`（或你本机 IP:3000）

### 2. 手机上访问
确保手机和本机在**同一 WiFi/局域网**下，打开 Chrome 浏览器访问：
```
http://10.250.39.42:3000
```

⚠️ **注意**：HTTP 方式下摄像头可用，但 PWA 安装和 Service Worker 不可用。功能测试不受影响。

---

## 方式二：HTTPS 访问（完整 PWA 体验）

### 1. 在本机启动 HTTPS 开发服务器
```bash
cd happy-rehab
npx vite --host 0.0.0.0 --port 3000 --https
```
首次启动会提示安装证书，确认即可（自签名证书）。

### 2. 手机上访问
```
https://10.250.39.42:3000
```

### 3. 手机安装自签名证书（一次性操作）
- Chrome 会提示"您的连接不是私密连接"
- 点击"高级" → "继续前往" 
- 然后在浏览器菜单中选择"添加到主屏幕"即可安装 PWA

---

## 方式三：离线文件部署（完全脱离电脑）

### 1. 把 `dist/` 目录复制到手机
将 `happy-rehab/dist/` 整个文件夹通过 USB 或文件传输工具复制到手机存储。

### 2. 手机上安装 HTTP Server 应用
在应用商店搜索安装：
- Android: "Simple HTTP Server" 或 "HTTP Server by Tautvydas"
- 免费，无需注册

### 3. 启动服务
- 用 HTTP Server 应用选择 `dist/` 文件夹
- 启动服务器（默认端口 8080）
- 手机浏览器访问 `http://localhost:8080`

同样，HTTP 下摄像头可用、PWA 不可安装。

## 常见问题

**Q: 摄像头画面黑屏？**
A: 检查浏览器设置 → 网站设置 → 摄像头权限 → 允许

**Q: 画面卡顿？**
A: 检查手机是否处于省电模式，关闭其他后台应用

**Q: 无法访问局域网地址？**
A: 确认手机和电脑在同一 WiFi，检查电脑防火墙是否拦截 3000 端口
